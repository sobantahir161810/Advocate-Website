## 🛠️ Customization

### Replace Contact Info
Search & replace throughout all HTML files:
- `+923160990069` → your WhatsApp number
- `contact@advocatejaveria.com` → your email
- `Peshawar` → your city (if different)

### Add Real Images
Replace SVG placeholders in HTML with `<img>` tags:
```html
<!-- Replace SVG block with: -->
<img src="assets/images/advocate-portrait.jpg"
     alt="Advocate Javeria — Legal Counsel Peshawar"
     class="welcome-img" loading="lazy">
```

**Recommended image sizes:**
- Hero portrait: 800×1000px (portrait)
- Welcome/About: 480×600px (portrait)
- Blog thumbnails: 400×220px (landscape)

### Add Favicon
```html
<!-- Add inside <head> on every page: -->
<link rel="icon" type="image/png" href="assets/icons/favicon.png">
<link rel="apple-touch-icon" href="assets/icons/apple-touch-icon.png">
```

### Go Live
1. Upload all files to your web host (maintain folder structure)
2. Update `sitemap.xml` with your real domain
3. Update `robots.txt` with your real domain
4. Submit sitemap to Google Search Console
5. Replace placeholder images with real photography
